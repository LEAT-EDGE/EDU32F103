Microcontroller

STMicroelectronics STM32F103VGT6

  • Core: ARM Cortex-M3
  • Frequency: 72 MHz (max.)
  • Flash: 1 MiB
  • RAM: 96 KiB

External connectors

  • USB Type-B (power, virtual COM port, JTAG programming and debugging, microcontroller as device)
  • 12-pin GPIO and power header

On-board peripherals

  • 8× push buttons
  • 8× slide switches
  • 16× LEDs
  • 4× 7-segment displays with decimal point
  • 1× Photoresistor
  • 1× LSM6DSL digital intertial measurement unit with 3-axis accelerometer and 3-axis gyroscope

Electrical specifications

  • Input voltage: 5V DC (USB connector)
  • Input current: 250 mA (max.)

Mechanical specifications

  • Size: 116mm × 76mm × 20mm

Supported firmware development tools

  • RTOS: Zephyr RTOS
  • Bare-metal (with HAL): STMicroelectronics STM32CubeMX
